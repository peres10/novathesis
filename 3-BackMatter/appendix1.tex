\chapter{Configuration Checklists by Institution}
\label{app:institution-checklists}

This appendix provides copy--paste snippets for the institutional presets included in the template.  
For each preset:
\begin{enumerate}
  \item Set the preset ID in \texttt{0-Config/1\_novathesis.tex} with \verb|\ntsetup{school=<id>}|.
  \item Open the corresponding \texttt{0-Config/9\_<...>.tex} file and uncomment \emph{exactly one} option block in each category (Department, Degree name, etc.).
  \item Recompile after each change; for backend/style changes run \verb|make clean| first.
\end{enumerate}

\bigskip
\noindent\textbf{Preset identifiers (as shipped):}
\begin{quote}\small
\verb|nova/fct|, \verb|nova/ims|, \verb|ulisboa/ist|, \verb|ulisboa/iseg|, \verb|ulisboa/fmv|, \verb|uminho|, \verb|uporto/fc|.
\end{quote}

\section{NOVA / IMS (\texttt{school=nova/ims})}
\label{app:nova-ims}

Open \texttt{0-Config/9\_nova\_ims.tex}. The file lists the UN Sustainable Development Goals (SDGs) with bilingual labels.  
Follow the in-file instructions to \emph{mark the SDGs relevant to your work}. If your program requires listing SDGs on the cover or abstracts, ensure your School’s rules are followed.

\end{verbatim}

\section*{General Reminders}
\begin{itemize}
  \item Keep exactly \emph{one} Department and \emph{one} Degree pair active when your preset provides catalogs (e.g., NOVA/FCT).
  \item For presets with optional metadata (classification, funding, embargo), include them only if required by your program.
  \item When switching bibliography backend or major preset options, run \verb|make clean| before recompiling.
\end{itemize}
